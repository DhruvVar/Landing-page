# Renesas Electronics on GitHub

Welcome to the official GitHub presence of Renesas Electronics.

This page is the main entry point to Renesas public software resources on GitHub. It is designed to help developers quickly navigate across our software offer, find the right starting point for their device family, and access the relevant repositories, documentation, examples, and support resources.

Renesas provides a broad software ecosystem spanning microcontrollers, microprocessors, tools, middleware, RTOS integrations, AI enablement, and reference software. This page organizes that offer in a simpler and more discoverable way.

Whether you are evaluating a new device, looking for board support, searching for software examples, exploring open-source resources, or trying to understand where to begin, this page is intended to guide you to the right location.

## Software Offer Overview

Renesas software on GitHub is part of a broader software ecosystem that includes:

- device software packages,
- board support and peripheral drivers,
- middleware,
- RTOS integrations,
- reference applications,
- documentation,
- and development tooling support.

The exact entry point depends on the product family and the intended development flow.

In general, users can think of the offer in the following way:

### 1. Family-Level Software Packages
These are the main software foundations for a device family.  
Examples include family software packages, driver packages, and device-centric software repositories.

### 2. Examples and Reference Applications
These help developers evaluate features, learn workflows, and start from working project baselines.

### 3. Documentation and Getting Started Material
These provide setup guidance, API references, usage instructions, and software architecture information.

### 4. Open Source and Ecosystem Integrations
These include GitHub-hosted integrations, community-facing repos, RTOS support, and selected cross-platform resources.

The RA software offer includes:

## How This GitHub Structure Is Intended to Work

This root page is intended to do three things well:

### 1. Explain the overall offer
At the top level, users should understand that Renesas software on GitHub is broader than a single repository or single product family.

### 2. Route users to the correct family/domain landing page
Instead of expecting users to search through the organization manually, this page should provide clear navigation by product family and software domain.

### 3. Route users from the landing pages to the actual implementation repos
The landing pages should then guide users to:
- the software package repository,
- the examples repository,
- the documentation,
- the release/download location,
- and the official Renesas product/software page.

This creates a cleaner and more scalable structure as the public software offer grows.
